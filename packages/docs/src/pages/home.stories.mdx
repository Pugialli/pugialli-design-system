# Pugialli UI - Design System


Um Design System é um conjunto organizado de padrões, componentes e diretrizes de design que garantem consistência e eficiência no desenvolvimento de produtos digitais. Como meu Design System, aqui irei encapsular a identidade visual e a experiência do usuário de maneira única, proporcionando diretrizes claras para garantir uma experiência coesa e intuitiva em todos meus produtos digitais.

No momento estas diretrizes estão associadas ao projeto desenvolvido no curso da Rocketseat, mas em breve teremos uma componentização adaptada para o meu projeto pessoal.
